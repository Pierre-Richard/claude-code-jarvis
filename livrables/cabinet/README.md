# Cabinet freelance

Ce dossier contient tous les livrables liés à ton activité freelance.

**Ce qui va ici :**
- Propositions commerciales et devis
- Contrats et modèles de documents
- Profils et présentations (LinkedIn, plateformes freelance)
- Supports de prospection (emails, messages, séquences)
- Fiches de mission et livrables clients
- Templates réutilisables (factures, onboarding, compte-rendu)

**Ce qui ne va pas ici :**
- Les documents clients que tu fournis en entrée (-> `context/import/`)
- Le code produit pour un client (-> `applications/` ou `sites-web/` selon le type)
