# Applications

Ce dossier contient tous les livrables liés aux outils, scripts et automatisations.

**Ce qui va ici :**
- Scripts (Python, Bash, PowerShell, Node.js...)
- Automatisations (workflows, bots, intégrations API)
- Outils CLI ou utilitaires
- Configurations et fichiers de setup

**Ce qui ne va pas ici :**
- Les sites et pages web (-> `sites-web/`)
- Les documents de ton activité freelance (-> `cabinet/`)
