# Sites web

Ce dossier contient tous les livrables liés à la création de sites internet.

**Ce qui va ici :**
- Sites vitrines et landing pages
- Portfolios
- Code HTML/CSS/JS de pages web
- Maquettes et wireframes textuels
- Contenu éditorial (textes, structure) pour un site

**Ce qui ne va pas ici :**
- Les outils et scripts (-> `applications/`)
- Les assets sources que tu fournis (-> `context/import/`)
